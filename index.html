Get Your Site Live

Follow these steps exactly. Takes about 15 minutes. You only have to do this once.

1
Take your plan to Claude and build your full site
Go to claude.ai and start a new conversation. Paste this opening message:

"I want to build my personalised Project:Rise escape plan site. Here is my plan summary: [paste your downloaded plan.txt here]. Please build me all the HTML files — daily routines, budget tracker, portfolio roadmap, outreach plan, destination checklist, and health routine — fully personalised to my situation."

Claude will build each page for you. Download every HTML file it produces and save them in one folder.
2
Create a free GitHub account
Go to github.com and sign up. This is where your files will live.
3
Create a new repository
Click the + icon top right → New repository. Name it project-rise. Set to Public. Click Create repository.
4
Upload your HTML files individually
On your repository page click Add file → Upload files. Drag all your HTML files in one by one or all at once — but not inside a folder. Make sure your homepage is named exactly index.html. Click Commit changes.
5
Deploy on Vercel
Go to vercel.com, create a free account. Click Add New → Project. Connect GitHub, select your project-rise repository, click Deploy. Your site will be live in 30 seconds at project-rise.vercel.app.
6
Bookmark it and start the plan
That's it. Open it every morning. Any time Claude updates a file for you, replace it in GitHub and Vercel redeploys automatically in 30 seconds. Now stop reading and start executing.
To update a file: go to your GitHub repo, click the file name, click the pencil edit icon, paste the new content, commit. Done.
